# Worker Test Studio - UX 개선 버전

## 변경 요약
- 어두운 네온/LLM 데모 느낌을 제거하고 흰색 배경 중심의 테스트용 UI로 변경했습니다.
- 테스트 설정 화면과 worker 화면을 더 명확하게 분리했습니다.
- 필수 입력 진행률, 입력값 자동 임시 저장, 초기화 버튼, 명확한 검증 메시지를 추가했습니다.
- 작업자 화면의 안내/지침/제출 흐름을 더 읽기 쉽게 정리했습니다.
- 기존 HTML ID와 JS 연결 구조는 유지했습니다.

## 실행
```bash
npm install
node server.js
```
브라우저에서 서버 주소로 접속하면 됩니다.
