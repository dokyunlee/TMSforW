Worker 테스트 메시지 설정

테스트용 작업 정보와 안내 기준을 입력하면 작업 시작 전·완료 후 메시지를 만들고, worker 화면에서 바로 검증할 수 있습니다.

1. 테스트 작업 입력제목, 보상, 지침, 작업 조건을 먼저 정리합니다.
2. 문구 확인·수정후보 문구를 비교하고 테스트용 최종 메시지를 직접 다듬습니다.
3. Worker 화면 검증테스트 링크로 worker 화면의 노출 방식과 진행 흐름을 점검합니다.

작업 기본 정보

작업자가 볼 정보와 메시지 작성에 필요한 맥락을 한 화면에서 정리합니다.

필수 입력 0 / 4

필수 항목을 채우면 바로 메시지를 만들 수 있습니다. 테스트 링크 생성 시 작업 정보가 테스트 저장소에도 기록됩니다.

작업자가 목록에서 바로 이해할 수 있는 짧은 제목을 입력합니다. worker 테스트는 사진 21장을 분류하는 작업으로 진행됩니다.

판단 대상과 행동을 한 문장으로 적습니다.

작업자 화면에 그대로 노출되는 내용입니다. 체크리스트처럼 작성하면 읽기 쉽습니다.

메시지 참고 정보 작업 부담과 테스트 맥락을 함께 입력하면 안내 문구가 더 자연스러워집니다.

운영 홍보 문구보다 테스트에서 검증할 활용 맥락을 담으면 신뢰도가 높아집니다.
